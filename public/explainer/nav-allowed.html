Feature allowed for the navigated iframe origin

To allow feature access to the origin that the iframe navigates to, list the origin in the iframe allow attribute. When using this syntax, the allowlist should include the 'src' token because the syntactic sugar where allow="geolocation" is a shorthand for allow="geolocation 'src'"

Response Header

Permissions-Policy:

geolocation=(self "https://pp-demo-trusted-site.glitch.me/" "https://pp-demo-trusted-iframe-nav.glitch.me/")

Trusted iframe tag

<iframe src="https://pp-demo-trusted-site.glitch.me" allow="geolocation 'src' https://pp-demo-trusted-iframe-nav.glitch.me/"></iframe>

Untrusted iframe tag

<iframe src="https://pp-demo-trusted-iframe-nav.glitch.me" allow="geolocation"></iframe>
